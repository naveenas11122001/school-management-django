# School Management Django

School Management Django is a web application built with Django that helps schools manage their day-to-day operations efficiently. This project provides features for managing teachers data in an organized way.

## Features
- Add, view, and manage teachers and students
- Assign students to classes and teachers
- Upload and manage media files (like teacher photos)
- Simple and responsive UI with templates and static files
- Built using Python and Django framework

## Tech Stack
- Python 3.x
- Django 4.x
- SQLite (development database)
- HTML, CSS (for frontend templates)

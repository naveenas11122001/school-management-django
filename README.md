# School Management Django

**School Management Django** is a web application built using the Django framework to help schools manage teachers, students, and classes efficiently. This project is ideal for small to medium-sized schools or as a learning project to understand Django development.

## Features
- **Teacher Management:** Add, view, and manage teacher profiles.  
- **Student Management:** Register students and assign them to classes.  
- **Class Management:** Organize students into classes and assign teachers.  
- **Media Handling:** Upload and display teacher photos or other files.  
- **Responsive Templates:** Clean and simple HTML/CSS templates for easy navigation.  
- **Extensible:** Can be extended to include attendance, grading, or notifications.

## Tech Stack
- **Backend:** Python 3.x, Django 4.x  
- **Frontend:** HTML, CSS  
- **Database:** SQLite (development), can be upgraded to PostgreSQL/MySQL  
- **Version Control:** Git & GitHub
